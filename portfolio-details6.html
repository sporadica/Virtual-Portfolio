Guided Actions in ADAPT

This e-learning was implemented to train both newly hired and seasoned associates on changes to an existing application. The program, ADAPT, is used to access timeshare owners' personal information, create new reservations, pay maintenance fees, etc. With the new feature, Guided Actions, associates can be guided through the procedure in real time, minimizing errors and time spent searching for procedural documentation.
